# Chrome Extension Icons

This folder should contain the extension icons in the following sizes:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create icons using:
1. Image editing software (Photoshop, GIMP, etc.)
2. Online icon generators
3. SVG to PNG converters

## Design Guidelines

- Use a simple, recognizable design
- Icon should represent text-to-speech (e.g., speaker icon, sound waves)
- Ensure good contrast for visibility
- Test on both light and dark backgrounds

## Temporary Solution

For development, you can use placeholder images or remove the icon references from `manifest.json`.
