# Zotero Plugin Icons

This folder should contain the plugin icons in the following sizes:

- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon64.png` - 64x64 pixels

## Creating Icons

Follow the same guidelines as the Chrome extension icons.

## Zotero-Specific Guidelines

- Icons should fit Zotero's professional, academic interface
- Consider using a book/document + speaker icon combination
- Test visibility in both light and dark Zotero themes
